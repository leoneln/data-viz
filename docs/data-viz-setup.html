Data Visualization: Setup Instructions

Welcome to the Data Visualization class! These instructions are for you to complete before our first day of class. Setting up your environment ahead of time will ensure we can dive right into our first assignment and hands-on exercise. Please follow the steps below to get started.

Summary of Required Tools

  • Posit Cloud Account:For Data Visualization with R.
  • VS Code + Python + Plotly:For Data Visualization with Python.
  • Power BI:Installed or online via school credentials.
  • Tableau Public Account & Desktop:Create account and install.

1. Set Up Posit Cloud (for R)

You will use Posit Cloud for the first assignment. Please follow these steps:

  1. Go to Posit Cloud.
  2. Create a free account (click on "Sign Up").
  3. Once your account is created, log in to Posit Cloud.
  4. Download the following files from GitHub for our first assignments: package_install.R, Hands On Exercise 1.qmd, and Hands On Exercise 2.qmd.
    • Please upload all three files to your new Posit Cloud project. You can do this now, but we will review the hands-on exercise files together in class.
    • Open the package_install.R script.
    • Run Part 1 of the script to install all the necessary libraries. This is the most important step to complete before class.
    • Part 2 of that same file is an optional (but fun!) exercise to get you acquainted with R and creating your first data visualization. I encourage you to try it if you have time, but it is not required.

Note: Don't worry about mastering the Posit Cloud UI; we will review it together in class. The main goal is to have your account created and the packages from Part 1 installed.

2. Set Up Python + Plotly

You will use VS Code, the latest version of Python, and the libraries of pandas, plotly, jupyter, and kaleido. I recommend installing the VS Code extensions Python, Pylance, Jupyter, and Black Formatter.

1. Install the Basics

  1. Install Python: Download from python.org/downloads. Windows users: make sure to check "Add Python to PATH".
  2. Install VS Code: Download from code.visualstudio.com. Launch VS Code and install the recommended extensions.

2. Create Course Folder & Virtual Environment

(This is optional, but highly recommended to isolate class packages).

  1. Create a folder, then open it in VS Code (File > Open Folder).
  2. Open a new terminal in VS Code (Terminal > New Terminal).
  3. Create a virtual environment (one time):
    Windows (PowerShell) 
    python -m venv .venv
.\.venv\Scripts\Activate
    OSX (Terminal) 
    python3 -m venv .venv
source .venv/bin/activate
    VS Code may prompt you to select this new environment. Select Yes.
  4. Install required libraries (make sure your virtual environment is active): 
    python -m pip install --upgrade pip
pip install pandas plotly jupyter kaleido black
  5. Why these libraries?
    • Pandas: for data wrangling.
    • Plotly: for creating interactive charts.
    • Jupyter: for VS Code’s interactive notebooks.
    • Kaleido: for exporting visuals (png, svg, pdf).
    • Black: to keep code formatting clean and consistent.

3. Quick Test (Jupyter Notebook)

  1. In VS Code, create a new Jupyter Notebook (File > New File > Jupyter Notebook) and name it hello_plotly.ipynb.
  2. In the first cell, type and run the following code: 
    import pandas as pd
import plotly.express as px

# tiny dataset
df = pd.DataFrame({
    "year": [2021, 2022, 2023, 2024, 2025],
    "enrollment": [25000, 26300, 27150, 26000, 29323]
})

fig = px.line(df, x="year", y="enrollment", title="Enrollment Over Time")
fig.update_layout(template="plotly_white")
fig.show()

4. Quick Test (Python Script)

If you prefer scripts, create a new file hello_plotly.py with this code:

import pandas as pd
import plotly.express as px

df = pd.DataFrame({"x":[1,2,3,4], "y":[3,1,4,2]})
fig = px.scatter(df, x="x", y="y", title="Hello, Plotly!")
fig.update_layout(template="plotly_white")
fig.show()

On the terminal, execute the file with:

python hello_plotly.py

3. Download and Setup Power BI

You will also need Power BI for class exercises. Here’s how to get it set up:

  1. Go to Office.com and log in with your school-provided credentials.
  2. Once logged in, search for Power BI in the Office apps.
  3. Download and install Power BI Desktop from the site if you are using a Windows machine. Otherwise, you’ll need to use Power BI Online.

4. Create a Tableau Public Account

For the next part of our course, we will use Tableau. Follow these instructions to get Tableau set up:

  1. Go to Tableau Public.
  2. Create a free Tableau Public account.
  3. After creating your account, download Tableau Desktop Public (this version is free).
  4. Install Tableau Desktop Public on your computer.
